🎵 Spotify Analytics Dashboard

 📌 Overview
This project is an interactive Power BI dashboard built using Spotify streaming data. It provides insights into track performance, artist popularity, album trends, and listening behavior through interactive visualizations.

---

 🎯 Objectives
- Analyze Spotify streaming data.
- Identify top-performing artists and tracks.
- Explore album popularity.
- Compare streams across different dimensions.
- Build an interactive dashboard for business insights.

---

 🛠 Tools Used
- Microsoft Power BI
- Power Query
- DAX
- Excel / CSV Dataset

---

 📊 Dashboard Features
- KPI Cards
- Interactive Slicers
- Top Artists
- Top Tracks
- Album Analysis
- Streaming Trends
- Dynamic Filters
